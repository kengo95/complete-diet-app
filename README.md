# アプリの概要

### タイトル

[食事管理アプリ](https://react-firebase-7e85f.web.app/)

### 開発経緯

前回作成した[Fai Fai Diet](https://fai-fai-di.web.app/)の改良版として本アプリを作成いたしました。

改善点としては
* 履歴を残せるようにする。
* 自分の食材を登録できるようにする。
* ユーザー別に認証する。

以上、３点を本アプリでは改善いたしました。
